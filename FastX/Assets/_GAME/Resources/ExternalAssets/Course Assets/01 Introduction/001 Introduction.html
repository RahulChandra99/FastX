Welcome to Make a driving game in Unity

If you get stuck at any point, or want to ask any questions, please leave me a message in the Q & A section, and I'll try and get back to you as soon as possible.

I would recommend creating this course in Unity High Definition Render Pipeline(HDRP), however, if your computer struggles with HDRP, then you can create the project in the Universal Render Pipeline (URP).

In Lecture 3 - Setting up the project - there are two resources:

Trees (These are for the HDRP)

Trees URP (These are only for the URP if you have set up that type of project)

You only need to download one of those sets of trees depending on the type of project you have created.
